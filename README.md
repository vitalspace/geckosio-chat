# Real Time Chat Example

A simple Real Time Chat using Geckos.io

```bash
# clone
npx gitget https://github.com/vitalspace/geckosio-chat.git

# change directory
cd geckosio-chat

# install
npm install

# run
npm start

# open
http://127.0.0.1:3000/
```
